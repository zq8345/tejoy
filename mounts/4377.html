1. Why Use a Junction Box

An outdoor junction box is a critical component for any permanent Starlink installation. It serves several essential functions:

  • Weather protection: Keeps cable connections dry and free from corrosion
  • Stress relief: Provides a secure anchor point for cables entering the building
  • Serviceability: Allows easy disconnection and reconnection for maintenance
  • Neat installation: Conceals unsightly cable splices and adapters
  • Fire safety: Contains any electrical issues within a fire-rated enclosure
Industry Standard: The National Electrical Code (NEC) requires that all outdoor cable splices and connections be contained in a weatherproof enclosure. The Tejoy Junction Box meets IP66 requirements for direct outdoor exposure.

4. Installation Guide

Tools Required

  • Drill with hole saw or step bit (for cable gland holes if using custom knockouts)
  • Screwdriver (Phillips)
  • Cable cutters / wire strippers
  • Caulk gun (for outdoor sealant)

Step-by-Step

Step 1: Choose the Location

Mount the junction box on an exterior wall near the Starlink cable entry point. It should be sheltered from direct rain if possible, though the IP66 rating allows direct exposure. Ideal height: 1.5 meters above ground for easy access.

Step 2: Mount the Box

Use the included mounting bracket or drill directly through the back panel mounting holes. Secure with appropriate wall anchors (included for masonry; use your own for wood siding). Ensure the box is level and the hinged lid opens downward for weather protection.

Step 3: Prepare Cable Entries

Open the desired knockout ports using pliers (twist to break the scored plastic). For the bottom entry, always use the lowest knockout to allow water drainage. Install cable glands into the knockouts and tighten the lock nut from inside.

Step 4: Route and Connect Cables

Feed cables through the cable glands into the junction box. Leave enough slack inside (30 cm minimum) for easy connection and future service. Make connections inside the box using weatherproof connectors. Do not leave bare wire exposed.

Step 5: Seal and Close

Tighten the cable gland compression nuts to seal around each cable. Apply a small bead of silicone sealant around the box-to-wall interface if mounting on an uneven surface. Close the lid and secure with the latch. Optionally, add a padlock for security.

Step 6: Test Connections

Power on the Starlink system and verify connectivity. Check that no cables are pulling or straining against the glands. Verify the lid closes fully without pinching cables.

5. Wiring Configurations

Configuration A: Simple Pass-Through

[Roof Cable] --> [Gland IN] --> [Junction Box] --> [Gland OUT] --> [Inside Wall]

Use for: Running continuous cable through the wall with a drip loop point and connection protection.

Configuration B: 12V Adapter Housing

[Terminal] --> [Gland IN] --> [12V Adapter inside JBox] --> [Gland OUT] --> [DC Power Source]

Use for: Housing the Tejoy 12V Power Adapter outdoors near the terminal. Reduces cable length between adapter and terminal (improves efficiency). Ensure the adapter is not in direct contact with the box interior floor — use standoff mounts for airflow.

Configuration C: Cable Extension Splice

[Terminal Cable] --> [Gland IN] --> [DC Coupler inside JBox] --> [Gland OUT] --> [Extension Cable to Power]

Use for: Joining two extension cables to reach longer distances. Use the Tejoy Waterproof DC Coupler (XLC-WDC-01) for a reliable outdoor splice.

6. Maintenance & Inspection

IntervalTask
MonthlyVisual inspection — check for cracks, water ingress inside box, or signs of corrosion
QuarterlyOpen the box and inspect connections for tightness; re-tighten cable gland nuts if needed
Bi-AnnuallyClean cable glands with a damp cloth; check gasket condition
AnnuallyReplace cable gland compression nuts if cracked or brittle; inspect sealant condition around mounting surface
Pro Tip: Place a small silica gel packet inside the junction box when closing it for the first time. This absorbs any residual moisture trapped inside during installation. Replace the packet annually.
