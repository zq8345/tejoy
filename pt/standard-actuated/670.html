Produtos/Standard Actuated
Categoria: Standard Actuated

Suporte para Starlink Gen 2, Suporte Pivotante

Fale conosco agora → Produtos

Projetado para instalação em telhados de telhas asfálticas (shingle), requer perfuração. Não recomendado para telhados de cerâmica, ardósia e barro. Para melhores resultados, o suporte deve ser ajustado para garantir que o Starlink seja instalado o mais próximo possível da vertical.

Compatibilidade: Standard Actuated Starlink, High Performance Starlink

Ferramentas necessárias: Furadeira, chave de soquete de 1/2" (12.7 mm), broca de 5/32" (4 mm)

Dimensões do suporte: Comprimento: 8.66" / 22 cm, Largura: 4.3" / 11 cm, Altura: 6.14" / 15.6 cm

Dimensões da embalagem: 9.7" x 5.5" x 3.1" / 24.7 x 14 x 7.9 cm

Peso da embalagem: 2.2 lbs / 1.0 kg
