Accesorios compatibles con Starlink para todas las generaciones de terminal.

Categoría: Mini

Adaptador Type-C a CC para Starlink Mini, Impermeable, Compatible con el Cable de Alimentación USB C Original del Starlink Mini, Fuente PD de 65W+, Adaptador de Energía CC

Contáctanos ahora Volver

  • Adaptador USB C a CC: Este adaptador USB C a CC para Starlink Mini está diseñado especialmente para convertir el conector CC del cable de alimentación CC del Starlink Mini en un conector USB-C, lo que te permite cargar tu Starlink Mini con un cargador USB-C o una batería portátil.

  • Impermeable: Este cable de alimentación del Starlink Mini es impermeable en el extremo hembra CC. Para lograr un mejor desempeño impermeable, puede costar trabajo insertar el cable de alimentación hasta el fondo del lado hembra CC; tal vez necesites girarlo mientras lo conectas para que quede bien acoplado.

  • Portátil y compacto: Este cable del Starlink Mini es pequeño y ligero, fácil de llevar a cualquier lado, y brinda una fuente de energía estable a tu antena Starlink Mini, además de soportar cualquier condición climática adversa que encuentres durante el uso.

  • Compatibilidad: Este cable del Starlink Mini es compatible con el cable original del Starlink Mini y funciona con un adaptador de pared PD de 65W+, una batería portátil o un cargador para vehículo, para que no te preocupes por buscar una fuente de energía adecuada para alimentar tu Starlink Mini.

  • Estabilidad y durabilidad: La capa exterior del cable de alimentación del Starlink Mini está hecha de PVC antioxidante de alta calidad, lo que brinda una alimentación estable a tu antena Starlink Mini. No importa a dónde te lleve tu aventura, la antena Starlink Mini transmitirá señales con eficiencia.
